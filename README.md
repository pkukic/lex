# lex
Lexical analyzer generator (simplified clone of lex) for a homework in Compiler Design at FER

## Installation
Run the install script to set up the environment:
```bash
./install.sh
```

## Usage
```bash
uv run python GLA.py < language_definition
```

## Testing
```bash
./test_lab.sh
# or
uv run python test_lab.py
```
